# AI Chat

## When to use

- 用户需要自由对话、多轮问答或调用通用 AI 能力时
- 需要结合知识库、记忆或工具结果来生成回答时
- 任务不属于某个更专门的 App 时，可先从 AI Chat 进入

## Guidelines

1. 优先保持对话连续性，回答前结合当前上下文和历史消息。
2. 涉及工具调用时，向用户展示关键过程，不要把工具执行变成黑盒。
3. 如果启用了记忆或知识库，尽量基于检索结果回答，并清晰区分事实与推断。
