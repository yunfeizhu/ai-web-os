# Browser

## When to use

- 用户要打开网页、查看链接或在系统浏览器中继续操作时
- 任务需要后续扩展成网页自动化、页面摘要或人工接管浏览时

## Guidelines

1. 把 Browser 当作真实网页会话的承载容器，而不是通用文本输出面板。
2. 能直接在浏览器里完成的网页查看行为，不要绕回文件管理器或聊天窗口模拟。
3. 遇到登录、验证码或高风险提交时，优先切换到人工确认或人工接管。
