# Notes

## When to use

- 用户要新建、编辑、整理或切换 Markdown 笔记时
- 任务更偏笔记写作、改写、扩写、总结，而不是通用文件编辑时

## Guidelines

1. 默认将笔记保存在 `/Notes`，保持该目录作为笔记主空间。
2. 保留 Markdown 结构，不要因为 AI 改写而破坏标题、列表、引用等语义。
3. AI 操作优先用于改写、扩写、提纲整理和总结，而不是无约束重写整篇内容。
