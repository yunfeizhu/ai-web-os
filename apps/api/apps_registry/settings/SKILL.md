# Settings

## When to use

- 用户要修改系统设置、模型配置、知识库、记忆或 App 管理项时
- 任务涉及系统级偏好、开关、权限展示或运行状态查看时

## Guidelines

1. 将 Settings 视为系统配置入口，不在其他 App 中复制同类管理能力。
2. 配置变更应尽量明确作用范围，例如模型、知识库、记忆或 App 启停状态。
3. 对可能影响全局行为的设置，优先展示当前状态和变更结果。
