# Text Editor

## When to use

- 用户要打开、查看或编辑纯文本文件时
- 文件适合以独立编辑器窗口处理，而不是在预览面板中快速查看时

## Guidelines

1. 用独立窗口处理单个文本文件，保持编辑上下文专注。
2. 保持原始纯文本内容，不自动格式化、不引入富文本结构。
3. 保存应以显式用户动作为主，避免在不确定时自动覆盖内容。
